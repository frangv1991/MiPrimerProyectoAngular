Filtro de fecha

Ejemplo:
Aplicando filtro dd/MM/yyyy HH:mm: {{ birthDate|date:'dd/MM/yyyy HH:mm' }}
Aplicando filtro fullDate: {{ birthDate|date:'fullDate' }}
Aplicando filtro shortDate: {{ birthDate|date:'shortDate' }}

Filtro de transformación de caractéres

Ejemplo:
Texto en mayúsculas: {{ dummyText|uppercase }}
Texto en minúsculas: {{ dummyText|lowercase }}

Filtro personalizado: stripHTML

Ejemplo:
Sin filtro: {{ htmlText }}
Con filtro: {{ htmlText|stripHtml }}

Filtro personalizado: truncate-string

Ejemplo:
Sin filtro: {{ dummyText }}
Con filtro: {{ dummyText|truncateString:{maxLength: 10, ellipsis: true} }}
